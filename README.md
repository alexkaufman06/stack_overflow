# Stack Overflow
This is a clone of stack overflow built with Ruby on Rails and bootstrap.
## Setup Instructions:
### Gems Used:
```
rails
pg
sass-rails
uglifier
coffee-rails
jquery-rails
turbolinks
pry
bootstrap-sass
shoulda-matchers
rspec-rails
capybara
```
### Install Bundler:
```
$ gem install bundler
```
### Run Bundler:
```
$ bundle
```
### Start the database:
```
$ postgres
```
### Create databases:
```
$ rake db:create
$ rake db:migrate
$ rake db:test:prepare
$ rails server
```
Type the url localhost:3000 into your web browser.
